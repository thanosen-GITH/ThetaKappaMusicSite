# Theta Kappa Music — Website Blueprint

## Site purpose

Theta Kappa Music is the central website for Thanos Karakantas’ work as a musician, Tonmeister, arranger, and developer of creative rhythm/audio tools.

The website should guide visitors into three main areas:

1. Music
2. Tonmeister
3. Apps / Software

The tone should be professional but personal. The site should not feel like three unrelated careers, but like one coherent artistic and technical profile.
